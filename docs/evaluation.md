# 검색 성능 평가

## 평가셋 구성

| 유형 | 쿼리 예시 | 정답 상품 | 비고 |
|---|---|---|---|
| 정확 매칭 | 나이키 에어맥스 270 | (상품 ID) | |
| 동의어/유사어 | 나이키 운동화 | 나이키 스니커즈 | 벡터 검색 강점 확인용 |
| 오타/문맥 | 나이키 런닝화 | 나이키 조깅화 | |

> 위 표를 20~30개 쿼리로 채워서 사용

## 측정 지표

- **Recall@5**: 상위 5개 결과 안에 정답 상품이 포함된 쿼리 비율
- **Recall@10**: 상위 10개 결과 안에 정답 상품이 포함된 쿼리 비율

## 케이스 스터디

- 벡터 검색이 BM25보다 우수했던 쿼리 사례 2~3개 (스크린샷/결과 캡처)
- 벡터 검색이 오히려 안 좋았던 사례 (한계점 기록)
